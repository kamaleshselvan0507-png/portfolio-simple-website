# portfolio-simple-website
This is a simple personal portfolio website built using HTML and CSS. It consists of two main pages: a Home page and a Contact page. The website is designed to present basic information about me and provide an easy way for visitors to get in touch.

The Home page highlights an introduction and overview, while the Contact page allows users to connect with me. The design focuses on simplicity, clean layout, and smooth navigation.

🚀 Features
Two-page structure (Home & Contact)
Clean and minimal design
Easy navigation between pages
Responsive layout for different screen sizes
🛠️ Built With
HTML5
CSS3
